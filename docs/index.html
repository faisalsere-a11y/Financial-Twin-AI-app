Trust by design

Know the data mode before you trust the answer.

The interface distinguishes sample, account, and advisor behavior instead of presenting them as the same thing.

Sample profile

Uses the included Saudi profile in SAR and does not connect to a bank.

SQLite-backed account mode

Uses the repository’s local server account flow. Production deployments still require institution-grade encryption, audit logging, and retention controls.

Advisor mode

Uses an optional server capability; deterministic scenario guidance remains available when that capability is disabled or unavailable.

FAQ

Clarity before commitment.

  • Is Financial Twin a bank or financial adviser?

    No. It is an educational simulation and planning layer. It does not hold funds, move money, or replace regulated advice.

  • Where do the numbers come from?

    The product uses deterministic cash-flow, debt, risk, health, goal, and investment engines. The sample journey uses the included Saudi profile in SAR.

  • What happens when advisor mode is unavailable?

    NOVA returns deterministic guidance derived from the scenario comparison, so the core decision workflow remains usable without an external advisor response.

  • Can I export my analysis?

    The reports area supports CSV export from the financial model. Every export should be reviewed before it is used for a real decision.
