Trust by design

Know the data mode before you trust the answer.

Sample mode: uses the included Saudi profile and does not connect to a bank.

Local server mode: uses the repository's SQLite-backed account flow. Production deployments still require institution-grade encryption, audit logging, and retention controls.

AI mode: uses OpenAI only when a key is configured and otherwise returns deterministic scenario guidance.

FAQ

Clarity before commitment.

Is Financial Twin a bank or financial adviser?

No. It is an educational simulation and planning layer. It does not hold funds, move money, or replace regulated advice.

Where do the numbers come from?

The product uses deterministic cash-flow, debt, risk, health, goal, and investment engines. The sample journey uses the included Saudi profile in SAR.

What happens when OpenAI is not configured?

NOVA falls back to deterministic recommendations derived from the scenario comparison, so the core product remains usable without an API key.

Can I export my analysis?

The reports area supports CSV export from the financial model. Every export should be reviewed before it is used for a real decision.
